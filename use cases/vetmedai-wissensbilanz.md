# VetMedAI Wissensbilanz - University Knowledge Balance Dashboard

## Summary

The VetMedAI Wissensbilanz is an interactive dashboard for exploring Austrian university performance metrics, developed as Use Case 1 within the VetMedAI project at Veterinarmedizinische Universitat Wien. The dashboard processes 76 Excel files covering 22 Austrian universities across 19 metrics. It was developed as a Promptotyping teaching case study, where workshop participants observed the development process directly.

The project produced the largest knowledge base in the Promptotyping corpus: 41 documents in a hierarchical 5-category structure. This includes a distinct Learnings subcategory (L001-L010) that distills transferable insights as a dedicated Process Document type. Four research hypotheses (H1-H4) about dashboard utility were formulated and documented as knowledge documents. The interface provides 7 visualisation types with dual-mode analysis, built with D3.js as a responsive dashboard. A tutorial system with badges supports the Promptotyping learning context.
