# coOCR-HTR - Browser-based OCR/HTR Workbench

## Summary

coOCR-HTR is a browser-based workbench for expert-in-the-loop transcription workflows. It renders OCR/HTR outputs alongside source images, enabling hybrid validation through deterministic rules and LLM-as-a-Judge. The prototype was completed in one workday using Promptotyping methodology and then consolidated.

The interface is a rapid prototype for understanding and validating a specific working process. The Promptotype is explicitly not a final product but a tool for testing whether a process works. It addresses epistemic asymmetry in LLM outputs by operationalising an "Editor-in-the-Loop" approach. A community fork has been integrated. The project awaits a comprehensive feature test and GitHub release v2.1.0.

## LLMs und Tools

- Claude Sonnet 4.5 (Claude Code) for development
- Vision-Language Models for validation
