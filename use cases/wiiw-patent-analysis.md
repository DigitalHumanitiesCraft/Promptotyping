# wiiw Patent Analysis - Network Analysis Dashboard

## Summary

Network analysis dashboard for international patent cooperation data (2010-2018), built during a data analysis workshop at the Wiener Institut fur Internationale Wirtschaftsvergleiche (wiiw). The project demonstrates Promptotyping applied outside Digital Humanities proper, using economic research data in CSV format. A 4-tab visualisation provides network overview (force-directed graph with centrality measures), temporal evolution (yearly snapshots), bridge countries (international cooperation patterns) and data methodology documentation. Static site deployed via GitHub Pages.

- Repository: [github.com/DigitalHumanitiesCraft/wiiw-patent-analysis-demo](https://github.com/DigitalHumanitiesCraft/wiiw-patent-analysis-demo)
