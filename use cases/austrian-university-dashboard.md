# Austrian University Dashboard

## Summary

Interactive dashboard for 76 Excel files containing Austrian university statistics. The project serves as the didactic baseline for Promptotyping video documentation, walking through every phase of the method: data exploration, data.md, knowledge documents (architecture.md, userinterface.md, userstories.md), overengineering check via a "Thinking Matrix", MVP simplification, milestone-based implementation, and screenshot feedback iteration. A Dune-themed redesign illustrates deliberate possibility space exploration.

- No public repository (video-documented only)
