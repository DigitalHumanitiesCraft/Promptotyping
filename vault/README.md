# Promptotyping Paper Vault

A [Grounded Vault](https://github.com/DigitalHumanitiesCraft/grounded-vault) instance that carries the provenance layer beneath the Promptotyping paper. The paper itself is maintained in this repository under `_content/paper/`; this vault anchors the paper's load-bearing claims (project figures, genealogy, novelty) to the source material that supports them, layer by layer, so that human expert review can proceed passage by passage.

Layer model, anchor mechanics, check contracts, and status progression are the template's invariants; see [SETUP.md](SETUP.md) for how this instance was parameterised and `knowledge/specification.md` for what this project decided. Humans start at [HOME.md](HOME.md), agents at [CLAUDE.md](CLAUDE.md).
