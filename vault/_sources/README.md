# _sources/

Original source files (PDF, PPTX, DOCX, CSV, …) live here. The folder is gitignored except for this file, because originals may be confidential or carry licenses that exclude redistribution. Whether an individual source may be versioned or published is recorded in the `license` and `confidential` metadata of its representation, see `knowledge/schema.md`.

Every file here is referenced by exactly one representation in `00_representation/`.
