# Minimal computing is a constraint-driven mode of DH praxis that resists equating innovation with scale.

## Statement

Risam and Gil define minimal computing as digital humanities work undertaken under a set of constraints, and as a mode of thinking about DH praxis that resists the idea that innovation is defined by newness, scale, or scope.
