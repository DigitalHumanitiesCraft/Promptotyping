# Community-curated tool registries on Wikidata answer the directory paradox by treating tool data as a maintained commons.

## Statement

Grallert et al. argue that curated tool directories inevitably go stale in fast-moving fields (the directory paradox), and propose open, community-curated registries that act as authority files and knowledge bases of a field's methodological practices, with Wikidata as the data store positioning tool data as part of the commons. This is the registry infrastructure the paper points to for the findability of small generated instruments.
