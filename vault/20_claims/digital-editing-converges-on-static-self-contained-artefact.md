# Digital scholarly editing arrives from its own sustainability discourse at the self-contained static web artefact as the durable publication form.

## Statement

Independently of any generative tooling, digital scholarly editing has reached the same target form the paper defaults to: a standalone static website whose pages exist as plain HTML needing no server-side execution or database, rendered from TEI-XML by deterministic XSLT templates, with longevity tied to the publishing institution rather than to a technology or a single developer. This is the paper's Section 4.1 evidence that the artefact type it proposes is the field's own answer to the longevity problem; the generation path differs, deterministic XSLT there and document-driven generation in a promptotype, while the target artefact agrees.
