# Coordinated and multiple views let users understand data through different linked representations.

## Statement

Coordinated and Multiple Views is an exploratory visualisation technique whose premise is that users understand their data better when they interact with it through several different, coordinated representations. Insight is formed through the interaction, which is why an exploration interface built on coordinated views functions as a primary research artifact rather than a presentation layer.
