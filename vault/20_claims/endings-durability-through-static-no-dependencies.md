# The Endings Principles secure digital longevity by building static sites free of server-side and external dependencies.

## Statement

Holmes and Takeda, reporting Project Endings, prescribe static standalone websites with no server-side software (no databases, PHP, or Python) and no dependence on external libraries or services, having rescued over a dozen projects from software obsolescence by reconstituting them this way.

## Support

- [[10_distillates/publications/holmes-2023-endings-principles#^s1]] — projects rescued by reconstitution as static standalone sites with minimal dependencies.

- [[10_distillates/publications/holmes-2023-endings-principles#^s3]] — no external libraries or services.
