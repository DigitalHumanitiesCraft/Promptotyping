# A research-database project applies Kemman's trading zone directly to data harmonisation, treating the negotiation of a shared language as a methodological core.

## Statement

A recent research-database project treats its data infrastructure as a methodological investment and runs data harmonisation as a trading zone, in which actors of different disciplines negotiate a shared model and description language to collaborate. This is a claim about how interdisciplinary knowledge can be stabilised that holds independently of any generative tooling, and it grounds the paper's Section 2.4 use of the trading zone: the site of interdisciplinary translation becomes a negotiated, shared vocabulary rather than an ad hoc conversation.
