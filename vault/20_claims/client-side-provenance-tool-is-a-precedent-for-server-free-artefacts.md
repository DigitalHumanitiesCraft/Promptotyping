# A same-venue provenance tool runs entirely in the browser without server-side architecture, a precedent for the server-free artefact type.

## Statement

A web-based provenance tool published in the same venue structures and publishes Linked Open Data as a client-side application that runs entirely in the user's browser, deliberately avoiding server-side architecture, hosting, database, and maintenance, which benefits resource-limited contexts. This grounds the paper's Section 4.1 footnote that a client-side precedent for the self-contained artefact type already exists in the field.
