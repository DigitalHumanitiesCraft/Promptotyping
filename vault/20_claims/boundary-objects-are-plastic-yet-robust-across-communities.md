# A boundary object adapts to each community's local needs while keeping a common identity across them.

## Statement

A boundary object is plastic enough to serve the different local needs of the communities that use it, yet robust enough to keep a common identity across their sites, and its creation and management is the key process by which coherence holds across intersecting social worlds. This grounds the paper's Section 2.4 move to read Promptotyping Documents as boundary objects between the scholar and the coding agent.
