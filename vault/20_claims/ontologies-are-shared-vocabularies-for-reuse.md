# A shared ontology is a common vocabulary that enables reuse of formally represented knowledge.

## Statement

Defining a common vocabulary for a shared domain of discourse supports the sharing and reuse of formally represented knowledge. In applied practice this holds concretely: the Bookkeeping Ontology's shared vocabulary eases communication within a project team and can be reused by other projects, so the model is criticised and extended through reuse.
