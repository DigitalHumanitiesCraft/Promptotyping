# Minimal computing means reducing computation literally: less code, fewer bytes, less processing.

## Statement

For Risam and Gil, reduction in minimal computing is meant literally, as less code or fewer bytes and, in turn, less computational processing time or capacity. This is the reductive commitment the paper draws on for the fewest-moving-parts artefact.
