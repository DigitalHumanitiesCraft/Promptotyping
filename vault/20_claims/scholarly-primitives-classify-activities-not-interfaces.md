# Unsworth's scholarly primitives classify research activities, not the epistemic work interfaces enable.

## Statement

Unsworth's scholarly primitives are basic functions common to scholarly activity across disciplines, over time, and independent of theoretical orientation, listed as discovering, annotating, comparing, referring, sampling, illustrating, and representing. They classify what researchers do, an activity taxonomy, rather than the epistemic function that a research interface performs. This is the distinction against which an interface typology by epistemic function is set.
