# Humanities data are constructed capta, actively taken, rather than given data.

## Statement

Drucker holds that humanistic knowledge is situated and constructed, so what looks like given data must be understood as capta, actively taken; the conventions built to express observer-independent knowledge need reworking to express interpretation.

## Support

- [[10_distillates/publications/drucker-2011-humanities-approaches#^s1]] — all data must be understood as capta and observer-independent conventions reworked.
- [[10_distillates/publications/drucker-2011-humanities-approaches#^s2]] — capta is taken actively while data is assumed given.
