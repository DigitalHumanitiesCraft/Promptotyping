# Endings-compliant artefacts function without JavaScript or CSS and require no ongoing tending, turning "Tamagotchis" into "Pet Rocks".

## Statement

The Endings aim is to convert high-maintenance "Tamagotchi" projects into zero-maintenance "Pet Rocks"; to that end every page should still function even without JavaScript or CSS support. Durability is achieved by removing what must be tended.
