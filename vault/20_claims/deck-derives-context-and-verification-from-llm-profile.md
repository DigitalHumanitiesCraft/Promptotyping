# The teaching material derives its practice consequences, curated context with verification and the harness argument, from the jagged-alien LLM profile.

## Statement

The workshop deck does not leave its LLM characterisation as description. As of the 2026-07-19 snapshot it draws two practice consequences from the jagged-alien profile. First, from the unpredictable capability profile it derives the need for curated context and verification, the need the workshop then develops. Second, it marks tool-use, context window and knowledge cut-off as system properties rather than capability jags, names tool-use as the first remedy that compensates some of the named weaknesses, and frames compensating model weaknesses through the environment as the harness argument. The claim records the derivation the material states and dates; it makes no assertion about whether the consequences are correct, which would require external sources.
