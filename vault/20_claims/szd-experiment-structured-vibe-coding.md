# The Stefan Zweig Digital experiment produced a tool in two hours and turned on reducing complexity.

## Statement

The first publicly documented Promptotyping experiment built a working timeline annotation tool for the Stefan Zweig Digital project in about two hours. Two moves are recorded as decisive: precise technical terminology steered the architecture, where "Single Page Application" triggers a single HTML file with embedded JavaScript and CSS while "Web App" would steer toward a React implementation; and the Critical Expert's key intervention was to reduce complexity, asking for a simple version without the AI support, collaboration, and visualisation the model had proposed.
