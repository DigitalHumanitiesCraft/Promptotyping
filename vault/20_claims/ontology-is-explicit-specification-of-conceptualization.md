# An ontology is an explicit specification of a conceptualization.

## Statement

In Gruber's canonical definition, an ontology is an explicit specification of a conceptualization, where a conceptualization is an abstract, simplified view of the world one wishes to represent for some purpose. The definition ties the formal artifact to a deliberate, purpose-driven abstraction of a domain.
