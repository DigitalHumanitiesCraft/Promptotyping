# The digital scholarly edition is itself an interface, addressed as GUI for humans and API for machines.

## Statement

Understanding a digital scholarly edition as an interface means understanding it as a connection point between documents and users, whether human or machine. Editions present two interface types accordingly, a Graphical User Interface for human readers and an Application Programming Interface for machine-to-machine exchange, so the edition mediates simultaneously to human and machine addressees.
