# Sacha's knowledge generation model describes the visual-analytics process by integrating existing frameworks.

## Statement

Sacha and colleagues propose a knowledge generation model that ties together diverse visual-analytics frameworks and gives a high-level description of the human and computer processes inside such systems. This grounds the paper's Section 5 use of the model as the process framework that comes closest to an epistemic account of visual analytics, yet describes the process rather than classifying interfaces by the epistemic function they serve.
